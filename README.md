# Adil Furniture

Welcome to the official repository for **Adil Furniture**.

🌐 Live Website: https://adilfurniture.vercel.app

## About

Adil Furniture is a modern furniture showcase website designed to present stylish home and office furniture products with an elegant and user-friendly online experience.

This repository contains the complete source code for the website.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* `index.html` — Homepage
* `about.html` — About Us page
* `gallery.html` — Product gallery
* `product.html` — Product listings/details
* `cart.html` — Shopping cart page
* `contact.html` — Contact page
* `style.css` — Styling and layout
* `script.js` — Interactive functionality
* `images/` — Website images and assets

## Features

* Responsive website design
* Multi-page navigation
* Product showcase interface
* Cart page structure
* Clean and modern UI
* Fast deployment with Vercel
